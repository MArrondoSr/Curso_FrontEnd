
Proyecto: Página de difusión y canal de contacto y ventas de Gráfica & Copiado (imprenta)

Se siguieron las consignas del curso, con la salvedad de que el formulario de contacto (a través de Formspree se reciben por mail las consultas) y las reseñas se ubican en páginas separadas a las que se accede mediante el menú de enlaces que se encuentra tanto en el encabezado como en el pie de página.
Para incorporar la sección de productos y el carrito de compras se usaron dos variantes. En una se toman los productos con todas sus características (imagen, descripción, precio, etc.) de una lista de objetos con productos reales (a modo de base de datos). En la otra se utiliza una API Fakestore, ya que esa es una de las consignas del programa del curso. La opción entre una u otra la hace el usuario al ingresar a la página de productos. La idea es descartar Fakestore más adelante y dejar solo los servicios reales del negocio. 
Todas las páginas del sitio se pueden visualizar correctamente tanto en pantallas desktop como en celulares. En este último caso el menú o barra de navegación se adapta al formato "menú hamburguesa" para mejorar la experiencia del usuario.
Las diferentes versiones del TP fueron guardadas en Github.
